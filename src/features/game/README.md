# Game Feature

Dieses Modul enthält die komplette Schnapsen/66 Spiellogik.

## Geplante Komponenten:

- **GameEngine**: Kern-Spiellogik (Regeln, Punkteberechnung)
- **CardDeck**: Kartendeck-Management
- **GameState**: Zustandsverwaltung während des Spiels
- **GameUI**: Spielfeld-Komponenten
- **AIOpponent**: KI-Gegner für Training

## Schnapsen-Regeln:

- 2 Spieler
- 20 Karten (Ass, 10, König, Dame, Bube)
- Ziel: 66 Punkte erreichen
- "Zudrehen" möglich
- Hochzeiten (König + Dame = 20/40 Punkte)
